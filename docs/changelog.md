# Changelog

Release history for the **ACME Azure** landing zone. Versions follow
[Semantic Versioning](https://semver.org) and are derived from
[Conventional Commits](https://www.conventionalcommits.org).

## How releases happen

Each release is cut from the commit history: `feat:` commits drive a
minor bump, `fix:` commits a patch bump, and a `!`/`BREAKING CHANGE`
footer a major bump. The changelog below is generated from those
commits.
