# Changelog

Release history for the **docs** component (the Zensical documentation
toolchain: `docs/`, `pyproject.toml`, `zensical.toml`, `Makefile`).
Versions follow [Semantic Versioning](https://semver.org) and are derived
from [Conventional Commits](https://www.conventionalcommits.org), tagged
`docs-v<version>`.

Other parts of the repository (environment configs under `configs/`,
Terraform modules under `modules/`) are versioned independently — see
their own `CHANGELOG.md`.

### How releases happen

Each release is cut from the commit history: `feat:` commits drive a
minor bump, `fix:` commits a patch bump, and a `!`/`BREAKING CHANGE`
footer a major bump. The changelog below is generated from those
commits.
